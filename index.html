Política de Privacidade — Meu Carro: Gastos e Manutenção 1. Informações Gerais Esta Política de Privacidade descreve como o aplicativo Meu Carro (com.meucarro.v2) coleta, usa e protege as informações dos usuários. Ao utilizar o aplicativo, você concorda com as práticas descritas aqui. 2. Coleta de Dados e Permissões O aplicativo solicita as seguintes permissões para funcionar corretamente: • Localização (GPS): Utilizada para capturar o ponto de partida e destino dos trajetos, calcular distâncias percorridas e permitir o funcionamento da "Jornada Automática". A localização em segundo plano é usada apenas se o usuário ativar explicitamente a função de Jornada. • Acesso a Notificações: Utilizada apenas se autorizada pelo usuário para capturar dados de aplicativos de logística e mobilidade (como iFood, Uber, 99) para preenchimento automático de trajetos. O app não lê notificações pessoais ou sensíveis. • Dados Financeiros: Todos os dados de gastos, despesas e combustíveis informados pelo usuário são armazenados localmente no dispositivo. 3. Armazenamento de Dados • Armazenamento Local: Os dados são salvos em um banco de dados interno no seu celular. • Backup no Google Drive: O aplicativo oferece a opção de sincronizar os dados com o seu Google Drive pessoal. Esses dados pertencem exclusivamente a você e não temos acesso a eles. 4. Serviços de Terceiros (Anúncios e Pagamentos) O aplicativo utiliza os seguintes serviços de terceiros que podem coletar dados para identificação do dispositivo: • Google AdMob: Para exibição de anúncios. Política de Privacidade do Google. • Google Play Services: Para mapas, localização e serviços base do sistema. • Google Play Billing: Para processar a compra da versão Premium. 5. Segurança Sua privacidade é nossa prioridade. Nenhum dado pessoal ou de localização é enviado para servidores externos de nossa propriedade. Todos os cálculos e registros permanecem sob seu controle total. 6. Alterações nesta Política Podemos atualizar nossa Política de Privacidade periodicamente. Recomendamos que você revise esta página ocasionalmente para verificar quaisquer alterações. 7. Contato Se você tiver dúvidas sobre esta política, entre em contato conosco através das informações fornecidas na loja.
